# Milestone 19: 遊戲架構重整 — 金錢/等級/升級曲線/任務主線+支線（6–15 歲台灣社會觀）

## Context
前一里程碑：18（家長審核佇列，complete）。
現況：經濟（M12 `ECONOMY`/`DIFFS`/`ALL_DONE_BONUS`）已成形，但**等級上限只到 50**（`awardXP`、`HUNTER_RANKS`、狀態顯示皆寫死 50），且沒有「主線敘事」——所有任務都是每日支線。使用者目標是 6–15 歲、能 0→100 成長、鼓勵「今天比昨天更好」的小型線上 RPG。

## Objective
把成長軸從 50 擴到 **100** 並定義可速通驗證的升級曲線；加入**主線（章節式成長敘事）**與**支線（每日家事/學習）**雙軌；補上「今天比昨天更好」的成長對比機制。全程不羞辱、不懲罰，貼合台灣家庭/校園。

## Scope
- **升級曲線 1→100**：`MAX_LEVEL=100`；`xpNeeded(level)=round(20+7L+1.2·L^1.5)`（前期快、後期穩），滿等總量約 8.4 萬 XP（每日全完成 ~235 XP ≈ 一年抵達）。`awardXP`/狀態列/XP 條的 50 上限改 `MAX_LEVEL`。
- **階級**：`HUNTER_RANKS` 擴為 E(1) D(16) C(31) B(46) A(61) S(76) 國家(91)。成就 a18「達到等級 50」文案修正（不再是「最高」）。
- **金錢架構**：沿用 `ECONOMY`（5🪙≈NT$1 僅換真實獎勵）。**來源**＝任務獎勵(DIFFS E12/D22/C28/B60/A140/S260)＋今日全完成(+25🪙/+20%EXP)＋連續加成＋主線章節獎勵；**去向**＝商店裝備(RPG 稀有度定價)＋真實獎勵(家長核准)。
- **主線 `MAIN_QUEST`**：10 章，依等級 1/5/10/20/31/46/61/76/91/100 解鎖，章節敘事貼合台灣成長情境並發金幣（mq1=0、累計 8400🪙）。`checkMainQuest()`（升級時前向發放、只發一次）、`backfillMainQuest()`（既有/遷移存檔標記已達章節為已領、不回溯補發避免通膨）、`mainQuestStatus()`。`player.mainQuestClaimed[]`。
- **支線**：既有每日家事/學習任務即支線（不變）。
- **成長對比**：`growthSnapshot()` 回傳今日/昨日完成數、個人最佳日(`counters.bestDayDone`)、是否達每日目標、是否超越昨天；狀態頁顯示主線當前章節＋「🔥今天比昨天更好」。

## Out of Scope
- 除錯/無限金幣速通工具與深度平衡調校（M20，本里程碑只提供 `progression.test.js` 基本速通驗證）。
- 排行榜（M21）、交易（M22）、真雲端（M23）。

## Acceptance Criteria
- [x] 金錢架構來源/去向明確；等級 1→100 曲線數值定義且可速通驗證（`progression.test.js`：84,028 XP、~358 天、單調遞增）。
- [x] 任務分主線（10 章成長敘事，台灣情境）+ 支線（每日家事/學習）。
- [x] 成長機制鼓勵「今天比昨天更好」：每日目標、個人最佳、超越昨天提示；無懲罰/羞辱設計。
- [x] 平衡可被 M20 速通驗證：到 Lv100 落在 180–720 天合理區間，無速成/卡關/通膨（mq 金幣前向不回溯）。
- [x] `test/progression.test.js` 10/10；smoke 77、cloud-store 9、accounts 8、parent-queue 7、desktop-save 4 全綠。

## Risk Notes
- **既有存檔**：`backfillMainQuest()` 在 `load()` 標記已達章節為已領、不補發金幣，避免高等玩家遷移時暴發金幣。
- **平衡**：~358 天到滿等為設計目標（長線陪伴）；M20 將以無限金幣/跳級工具做更細的階段收支與卡關分析後微調。
