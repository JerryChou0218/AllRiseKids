# KidQuest：覺醒 ⚔️ 獵人任務系統（v3）

一款給 8–12 歲孩子的每日任務 RPG 網頁遊戲，採用《我獨自升級》風格的暗黑系統介面：
完成現實生活任務 → 獲得 EXP / 金幣 → 升級配點、解鎖技能樹、攻略傳送門、兌換獎勵。

## 🚀 如何開始

直接用瀏覽器打開 `index.html` 即可遊玩，零安裝、零建置。
所有進度自動儲存在瀏覽器 localStorage（鍵名 `kidquest_data`）。

### 📲 安裝為 App（PWA，可離線）
透過本機伺服器開啟（PWA 必須走 http/https，不支援 `file://`）：
```
# 在 KidQuest 資料夾執行任一種：
python -m http.server 8000      # 然後瀏覽 http://localhost:8000
npx serve .                      # 或用 Node
```
開啟後，瀏覽器選單會出現「**安裝 / 加入主畫面**」。安裝後：
- 桌面/主畫面有獨立 App 圖示，全螢幕無網址列。
- Service worker 會快取 `index.html` 與 Tailwind/字型等 CDN 資源，**首次上線載入後即可完全離線遊玩**（解決原本斷網會掉樣式的問題）。
- 改版時更新 `sw.js` 的 `CACHE_VERSION` 即可讓使用者取得新版。

### 🖥️ 單機桌面版（.exe，1080P FHD 視窗，存檔隨資料夾移動）
不需瀏覽器、不需網路設定，雙擊資料夾內的 `KidQuest.exe` 就能玩（**1920×1080 視窗模式**、可全螢幕）。
- **寬版 PC 佈局**：桌面（≥1024px）採左側欄主導覽 + 寬版內容區（行動裝置仍為底部導覽的直式佈局），同一份 `index.html` 自動切換。
- **頂部列**：視窗最上方常駐帳號（登出/切換）與家長登入。
- **家長模式**才會看到台幣估值與月預算（`本月已核准真實獎勵 NT$ = 本月家長按下「核准」的兌換申請其 estimatedTwd 總和`）；孩子畫面只顯示金幣與兌換次數。
**帳號與進度存成 `.exe` 同層的 `kidquest-save.json`** —— 把整個資料夾複製到別台電腦，存檔跟著走、進度不遺失。

開發 / 打包（需先 `npm install`）：
```
npm start        # 開發模式直接執行桌面版
npm run pack     # 打包成 dist/KidQuest-win32-x64/（內含 KidQuest.exe）
npm test         # 69 項邏輯測試（另有桌面存檔、多帳號測試）
```
- 打包後把 `dist/KidQuest-win32-x64/` 整包資料夾交給使用者即可，存檔產生在這個資料夾內。
- 桌面版的存檔後端會自動切換：在 `.exe` 內寫檔案、在純瀏覽器內用 localStorage（同一份 `index.html` 兩用）。
- 註：桌面版的介面樣式（Tailwind / 字型）目前仍走 CDN，**首次需連網**；若要在完全離線的電腦上也保持樣式，需把這些資源在地化（可另開里程碑處理）。

### 👥 本機多帳號（登入 / 登出 / 密碼 / 切換）
多個孩子可在同一台裝置各自存進度：
- 啟動時若已有帳號 → 顯示**登入畫面**，選帳號、輸入密碼（可留空）即可進入。
- 管理者模式 →「帳號管理」可**設定/變更本帳號密碼**、**登出 / 切換帳號**、刪除非登入中的帳號。
- 在登入畫面可**建立新帳號**（接著走覺醒引導建立角色）。
- 資料配置：活躍帳號的即時進度存於 `kidquest_data`（桌面版即 `kidquest-save.json`）；帳號清單與各帳號快照存於 `kidquest_accounts`（密碼以雜湊保存，非明文）。各帳號進度完全獨立。
- 尚無帳號時沿用原本單人流程；既有單人存檔首次啟動會自動轉成第一個帳號。

v1 / v2 / v3 舊存檔會自動遷移到 v4，保留等級、金幣、能力點、技能、裝備、成就、家長 PIN 與紀錄。

## 🎮 玩法流程

1. **覺醒**：首次啟動登錄獵人名稱、**年齡（5–15）與性別**、選擇職業（🔮 魔法師 / ⚔️ 戰士 / 🛡️ 守護騎士，各有任務 EXP +50% 加成）
2. 系統每天自動從**任務池（100 個任務）**依年齡/性別抽出 **8 個今日任務**（E/D/C/B 四級平均分配），A/S 級挑戰每週抽 2 個
3. 孩子完成現實任務後按「挑戰完成」→ 進入「攻略確認中」
4. **管理者模式**（狀態頁 🔒，預設 PIN：`1234`）核准後發放報酬
5. 完成全部每日任務 → **今日攻略完成** 特效＋額外 25 金幣＋當日任務 EXP +20%＋連續紀錄 🔥+1（S級不計入門檻）

## ⚙️ 核心系統

| 系統 | 說明 |
|---|---|
| 任務池 | **100 個任務、8 大分類**：學校教育 / 家庭教育 / 品格修養 / 生活能力 / 情緒管理 / 健康習慣 / 人際互動 / 創作表達。每個任務含 `ageRange`（適齡）、`estimatedMinutes`、`why`、`repeatRule`，內容貼近台灣家庭與學校生活、鼓勵成長不懲罰 |
| 6 級難度 | **E**(易,3分) / **D**(普,3–10分) / **C**(中,10–25分) / **B**(難,25–60分) / **A**(跨日週挑戰) / **S**(大型挑戰)。E/D/C/B 進每日抽選，A/S 為每週挑戰 |
| 年齡 / 性別 | 覺醒時與管理者模式皆可設定孩子**年齡（5–15）與性別**，任務依此提供合適難度與內容 |
| 成長點數 | 每升級 +3 能力點自由配點（力量/敏捷/智力/活力/感知），各有實際效果 |
| 技能樹 | 每升級 +1 技能點，三職業各 7 技能依序解鎖 |
| 傳送門地圖 | 今日任務化身發光傳送門，可依 E/D/C/B/A/S 篩選；A/S 級 DUNGEON BREAK |
| 連續紀錄 | 只計每日任務（A/S 不影響）。加成拆兩套：**EXP 每天 +10%（上限 +100%）、金幣每天 +5%（上限 +50%）**；漏一天即歸零（無保護券） |
| 商店（虛擬道具）| **58 項裝備道具**（含 5 大套裝 + 擴充裝備）＋類型篩選＋感知折扣。**RPG 稀有度定價，不顯示台幣**：普通約 1–2 天、稀有 3–5 天、史詩 1–2 週、傳說 3–5 週 |
| 真實獎勵 | **預設 25 個（小/中/大三級）**。**以台幣家庭預算定價（5 金幣 ≈ NT$1）**，兌換採「申請 → 家長核准/退回退款」流程；**孩子畫面只顯示金幣與約需天數，不顯示任何台幣金額** |
| 視窗頂部列 | 視窗最上方常駐：左＝**帳號（點擊登出/切換）**、右＝**🔒 家長登入**；不需再從畫面內進入 |
| 賣回商店 | 背包裡不要的裝備可**賣回商店，返還原價值的一半**（💰 賣出） |
| 我的密碼 | 同一台裝置可能多人共用，使用者可在狀態頁**自設密碼保護自己的帳號** |
| 成就 | 20 個獵人成就，解鎖送金幣 |

## 👨‍👩‍👧 管理者（家長）功能

- 待審核任務核准 / 退回
- **任務池管理**：每個任務可啟用/停用、設定排程（每日/每週S/週末限定）、建議年齡（全年齡/8–10/10–12）、編輯/刪除/新增
- **今日任務設定**：每日任務數量（4–12）、孩子年齡層、🎲 一鍵重新抽選今日任務
- **待處理真實獎勵申請**：核准發放 / 退回（退回自動退還金幣）
- **真實獎勵管理**：新增、**編輯（名稱 / 所需點數）**、刪除
- **帳號管理**：對**任何帳號**變更密碼（🔑）或刪除（🗑️，含登入中帳號）
- **經濟摘要**：金幣 ≈ 台幣比例、月度真實獎勵預算（NT$900）、本月已核准估值、超預算提醒（僅提醒不限制）
- **變更家長 PIN 碼**、本週統計、完整紀錄
- （已移除「重置所有資料」按鈕，避免誤觸清空進度）

## 💰 經濟設計

集中設定於 `ECONOMY` 物件。兩套獨立邏輯：

- **真實獎勵**：以台灣家庭預算定價，`5 金幣 ≈ NT$1`、`cost = estimatedTwd × 5`。建議月度約 NT$700–1,000。每日認真完成約 120–180 金幣、一週約 900–1,400 金幣（≈ NT$180–280/週）。
- **虛擬道具 / EXP**：純 RPG 成長曲線，不顯示台幣。道具依稀有度與戰鬥力定價；EXP 用 `xpNeed(level)` 曲線（前期 1–2 天升級、中期 3–5 天、後期 1–2 週）。
- **任務報酬**：E 12EXP/8🪙、C 28/18、B 60/35、S 140/100。每日 3E+3C+2B 基礎約 148 金幣。

## 🛠️ 技術

- 單一 HTML 檔（Vanilla JS ES6+），CSS 發光特效 + emoji，無外部圖片
- Tailwind CSS CDN + Google Fonts（Noto Sans TC / Rajdhani）
- 響應式：手機 375px ～ 桌機 1200px
- `refs/` 為介面參考圖（非執行所需）；`test/smoke.js` 為煙霧測試

## ⚠️ 注意

- 清除瀏覽器資料會清除遊戲進度
- 進度為純本機儲存、不會自動同步。換瀏覽器 / 換電腦時，可在**管理者模式 → 設定**用「📤 匯出存檔」存成檔案，於新裝置「📥 匯入存檔」延續進度（純本機檔案，不上傳、不需帳號）。
